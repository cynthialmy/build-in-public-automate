# X (Twitter) Posting Skill

## Constraints
- Max 280 characters per tweet
- Use `threadParts` array for content that needs multiple tweets
- Each thread part must also be under 280 characters

## Strategy
- Lead with the most interesting metric or outcome
- Prefer concrete numbers and specifics over vague claims
- One idea per tweet — don't cram
- Use line breaks for readability

## Hooks
- End with a question or invitation to reply
- "What's your approach to [topic]?"
- "Anyone else hit this?"
- "Shipping this today — thoughts?"

## Thread Tips
- First tweet should stand alone as a compelling post
- Number threads naturally: don't use "1/n" unless 4+ tweets
- Last tweet: recap or CTA

## Avoid
- Generic hashtags (#coding #developer) — use specific ones sparingly if at all
- "Excited to share" openers
- Walls of text in a single tweet
